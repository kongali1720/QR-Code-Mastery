# Contributing to QR Code Learning
Thank you for considering contributing! We welcome improvements, corrections, and new educational content.


## How to Contribute
1. Fork the repository
2. Create a new branch (`git checkout -b feature-name`)
3. Commit your changes (`git commit -m "Add feature"`)
4. Push the branch
5. Open a Pull Request


## Contribution Guidelines
- Content must be ethical, legal, and educational
- No harmful code or exploitation tools
- Keep documentation clear and accessible
- Follow the existing folder structure
- Be respectful and constructive
